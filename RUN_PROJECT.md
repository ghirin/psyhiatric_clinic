# Психиатрическая больница - Запуск проекта

## Быстрый запуск

```bash
cd /app

# 1. Проверка системы
python check_system.py

# 2. Применение миграций (если есть новые)
python manage.py migrate

# 3. Запуск сервера
python manage.py runserver 0.0.0.0:8000
```

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `N` | Новый пациент |
| `S` | Поиск пациента |
| `C` | Календарь |
| `E` | Экспорт |
| `H` | Дашборд |
| `?` | Справка |

## Структура файлов

```
patients/
├── models.py           # Модели Patient, Hospitalization, Diagnosis
├── views_class_based.py # Основные представления
├── views_calendar.py   # Календарь, быстрые действия
├── forms.py            # Формы
├── urls.py             # URL маршруты
├── mixins.py           # Миксины
├── validators.py       # Валидаторы (ИНН, МКБ, паспорт, телефон)
├── mkb_data.py         # База кодов МКБ-11 (200+ кодов)
├── audit.py            # Логирование аудита
├── encryption.py       # Шифрование данных
└── migrations/         # Миграции
```

## Логирование аудита

Модуль `audit.py` автоматически отслеживает:
- Создание пациентов
- Просмотр карточек
- Редактирование данных (с историей изменений)
- Удаление
- Экспорт
- Загрузка документов

## Шифрование данных

Модуль `encryption.py` шифрует чувствительные поля:
- Диагнозы
- Анамнез
- Примечания

Для активации:
1. `pip install cryptography`
2. Добавьте `ENCRYPTION_KEY` в `settings.py`
